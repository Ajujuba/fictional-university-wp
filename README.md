<!-- First Steps -->
- Download the project and DB
- Run 'composer install'
- Create a .env and configure access to your DB (You can see env.example)
- Run ‘npm install’ inside the custom theme folder ‘Fictional University Theme'
- Configure your package.json

<!-- To edit style or js of theme: -->
- Open the Theme folder and run 'run npm start' to listen your changes in scss and js files

OBS: To edit the plugins:
     you need run 'npm install' inside the plugin folder
     After, configure your package.json
     run 'npm run start'
